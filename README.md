# YouTube Clone

A simple YouTube homepage clone built using only **HTML** and **CSS**.  
This project focuses on recreating the layout and styling of YouTube’s user interface.

## Preview

The project includes:

- Responsive navigation bar
- Sidebar menu
- Video grid layout
- Search bar UI
- Channel/profile sections
- Modern YouTube-inspired design

## Technologies Used

- HTML5
- CSS3

## Project Status

This project currently contains only the **frontend structure and styling**.

JavaScript functionality such as:

- Video playback
- Search functionality
- Sidebar interactions
- Dynamic content loading
- Dark mode toggle
- API integration

will be added in future updates.

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ay0m1de24/Youtube-clone.git
```

2. Open the project folder:

```bash
cd Youtube-clone
```

3. Run the project by opening `index.html` in your browser.
